# Imagens

Coloque aqui as imagens do bolão. Para trocar, basta substituir o arquivo mantendo o mesmo nome.

- `copa.svg` — símbolo exibido no topo da página (troféu/ícone da Copa). Substitua por SVG ou PNG do ícone da FIFA/Copa. Se usar outro formato, ajuste o `src` em `public/index.html` (`<img class="fifa-trophy" ...>`).
- `logo.svg` (opcional) — logo da NAVEPRO no cabeçalho. Se o arquivo existir, ele aparece no lugar do texto.

Bandeiras/escudos dos times são definidos por partida no painel do organizador (campo de bandeira aceita URL de imagem ou emoji).
